## Furniture Mall Website System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/0caae9bd74c543f7a428362b1f0d2050/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Merchant, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Merchant Information
5. Furniture Information
6. Furniture News
7. Personal Center
8. My Orders
9. My Addresses
10. My Favorites
11. Shopping Cart
12. Change Password
13. My Favorites Management
14. Order Management


**Merchant Functions:**
1. Login
2. Change Password
3. Personal Information
4. Merchant Management
5. Furniture Information Management




**Administrator Functions:**
1. Login
2. Change Password
3. User Management
4. Merchant Management
5. Furniture Information Management
6. Carousel Management
7. Furniture News Management
8. Order Management
